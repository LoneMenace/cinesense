CineSense is a Streamlit-based web application that analyzes the sentiment of movie reviews using a machine learning model trained on the IMDb dataset.

## Features
- Sentence-level sentiment analysis
- Confidence scoring
- Overall sentiment aggregation
- Visual sentiment distribution
- Model explainability (top contributing words)

## Tech Stack
- Python
- Streamlit
- scikit-learn
- AWS EC2 (deployment)

## Deployment
The application is deployed on an AWS EC2 instance and accessible via the instance public IP.

## Future Improvements
- Dockerized deployment
- CI/CD with GitHub Actions
- Persistent logging using Amazon S3
